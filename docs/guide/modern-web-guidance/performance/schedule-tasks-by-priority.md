# Планирование задач по приоритету

При разработке сложных веб-приложений задачи имеют разный уровень срочности. Завершение задач для текущего отображения является более важным, чем отправка аналитики или предварительное получение (prefetching) ресурсов. Prioritized Task Scheduling API позволяет планировать работу с определенными приоритетами, обеспечивая отзывчивость браузера на ввод пользователя.

## Реализация

Используйте `scheduler.postTask()` для планирования задач с одним из трех приоритетов:

- `user-blocking`: задачи, блокирующие пользовательское взаимодействие (например, обработку ввода, критичный рендеринг).
- `user-visible`: задачи видимы пользователю, но не являются блокирующими (дефолтное значение).
- `background`: задачи не видны пользователю и не являются блокирующими (например, аналитика, предварительное получение ресурсов).

```javascript
// Планируем высокоприоритетную задачу, блокирующую основной поток
scheduler.postTask(() => {
  // Выполнение критичного обновления, влияющего на опыт пользователя
  handleCriticalUpdate();
}, { priority: 'user-blocking' });

// Планирование задачи с приоритетом по умолчанию
scheduler.postTask(() => {
  // Рендерим некритичный контент, видимый пользователю
  renderSecondaryContent();
}); // 'user-visible' по умолчанию

// Планируем низкоприоритетную фоновую задачу
scheduler.postTask(() => {
  // Выполняем тяжелую фоновую работу, время выполнения которой не критично
  sendAnalytics();
}, { priority: 'background' });
```

## Поддержка и страховка

Scheduler API имеет ограниченную поддержку браузеров. Поддерживается в Chrome 129+, Edge 129+ (с сентября 2024 года) и Firefox 142+ (с августа 2025 года).
Не поддерживается в Safari.

Для поддержки приоритизации задач в старых браузерах нужно использовать такой полифилл:

```javascript
// Определяем поддержку Scheduler API
if (!('scheduler' in window && 'postTask' in window.scheduler)) {
  // Условно загружаем полифилл для браузеров без поддержки
  const script = document.createElement('script');
  script.src = 'https://unpkg.com/scheduler-polyfill';
  script.onload = () => {
    // Полифилл загружен и готов к использованию
    runScheduledTasks();
  };
  document.head.appendChild(script);
} else {
  runScheduledTasks();
}

function runScheduledTasks() {
  // Теперь можно безопасно использовать scheduler.postTask() во всех браузерах
  scheduler.postTask(() => {
    console.log('Task with priority support');
  }, { priority: 'background' });
}
```
