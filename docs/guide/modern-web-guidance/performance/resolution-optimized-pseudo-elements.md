# Использование изображений с оптимизированным разрешением в псевдоэлементах CSS

Использование изображений с оптимизированным разрешением в псевдоэлементах CSS (таких как `::before` или `::after`) позволяет добавлять декоративные иконки или структурную графику без загромождения HTML дополнительными узлами DOM. Сочетая псевдоэлементы с функцией CSS `image-set()`, можно предоставить браузеру несколько форматов (таких как AVIF или WebP) и разрешений (например, `1x` и `2x`). Браузер сам выберет наиболее оптимальное для устройства пользователя изображение.

## Реализация

Функция `image-set()` может использоваться прямо со свойством `content` псевдоэлемента или внутри его свойства `background-image` (при установке `content: ""`). Хотя предоставление формата изображения через `type()` и разрешения (например, `1x` или `2x`) дает лучший результат, оба аргумента являются опциональными.

```css
.icon-button::before {
  /* Используем image-set() прямо в свойстве content. */
  /* Всегда упорядочивайте форматы от наиболее оптимизированного (AVIF) к наименее (JPEG/PNG).
     Браузер остановится на первом поддерживаемом формате. */
  content: image-set(
    url("icon.avif") type("image/avif") 1x,
    url("icon-2x.avif") type("image/avif") 2x,
    url("icon.webp") type("image/webp") 1x,
    url("icon-2x.webp") type("image/webp") 2x,
    url("icon.png") type("image/png") 1x,
    url("icon-2x.png") type("image/png") 2x
  );

  display: inline-block;
  margin-right: 8px;
  vertical-align: middle;
}
```

## Поддержка и страховка

Функция `image-set()` поддерживается всеми основными браузерами с 18.09.2023.

Для старых браузеров, не поддерживающих `image-set()`, нужно предоставить стандартное определение изображения перед `image-set()`. Эта стратегия прогрессивного улучшения полагается на каскадную природу CSS: неподдерживаемые правила игнорируются.

```css
.icon-button::before {
  /* Резерв для браузеров без поддержки image-set() */
  content: url("icon.png");

  /* Современные браузеры будут применять это */
  content: image-set(
    url("icon.avif") type("image/avif") 1x,
    url("icon-2x.avif") type("image/avif") 2x,
    url("icon.png") type("image/png") 1x,
    url("icon-2x.png") type("image/png") 2x
  );
}
```
