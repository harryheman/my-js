# Сжатие шапки страницы при прокрутке

Сжатие (shrinking) шапки страницы при прокрутке - популярный шаблон UI, когда фиксированный элемент шапки наверху страницы плавно анимируется (transition) к меньшему размеру при прокрутке пользователем страницы вниз. Этот эффект часто используется, чтобы максимально увеличить площадь экрана для основного контента, сохраняя при этом доступность важных элементов навигации или бренда. С помощью анимации, управляемой прокруткой, этот эффект можно реализовать декларативным и производительным способом, привязав анимацию к положению прокрутки документа.

## Реализация

Вот как реализовать сжатие шапки при прокрутке:

1. Создаем фиксированную шапку: начинаем с элемента шапки, который фиксируется наверху страницы и имеет фиксированную высоту.

```html
<header>Шапка</header>
```

```css
header {
  position: fixed;
  height: 200px;
  top: 0;
  left: 0;
  right: 0;
}
```

2. Определяем анимацию сжатия: создаем анимацию CSS, изменяющую высоту шапки:

```css
@keyframes shrink {
  to {
    height: 50px;
  }
}
```

3. Применяем анимацию и временную шкалу прокрутки (scroll timeline): добавляем анимацию к шапке и используем функцию `scroll()` для привязки ее к положению прокрутки документа:

```css
header {
  animation: shrink auto linear both;
  animation-timeline: scroll(block root);
}
```

4. Устанавливаем `animation-range`: используем свойство `animation-range` для определения расстояния прокрутки, на котором должна происходить анимация. Например, для сжатия шапки на первых 150 пикселях прокрутки нужно использовать `animation-range: 0px 150px;`:

```css
header {
  animation-range: 0px 150px;
}
```

Подсказка: для предотвращения перекрытия шапкой следующего за ним контента добавьте свойство `padding-top` к `body` (или контейнеру основного контента) со значением, равным начальной высоте шапки.

Подсказка: для того, чтобы контент прокручивался синхронно со сжатием шапки, значением `animation-range-end` должна быть разница между начальным и конечным размерами шапки. Это гарантирует, что анимация завершается точно при достижении шапкой своего конечного размера. В нашем случае шапка сжимается от `200px` до `50px`, поэтому свойство `animation-range-end` устанавливается в значение `150px`.

## Пример кода

```css
@keyframes shrink {
  to {
    height: 50px;
  }
}

header {
  animation: shrink auto linear both;
  animation-timeline: scroll(block root);
  animation-range: 0px 150px;
}
```

## Лучшие практики

При использовании анимации, управляемой прокруткой, важно следовать нескольким лучшим практикам для обеспечения плавного и доступного (accessible) опыта:

- Используйте определение поддержки: не все браузеры поддерживают анимацию, управляемую прокруткой. Используйте `@supports ((animation-timeline: view()) and (animation-range: entry))` для определения поддержки и предоставляйте резерв для браузеров без поддержки.
  - Проверка `(animation-range: entry)` предназначена для исключения браузеров с частичной поддержкой.
  - Не используйте пакет `scroll-timeline-polyfill` в качестве резерва, поскольку в нем реализован не весь функционал, и он содержит много известных проблем.
  - Если анимация является чисто декоративной, резерв можно не предоставлять.
- Учитывайте предпочтения пользователей: некоторые пользователи предпочитают минимум движений в вебе. Используйте медиа-запрос `prefers-reduced-motion` для отключения или уменьшения количества анимаций для таких пользователей.
- Старайтесь анимировать только "производительные" свойства CSS: для плавной анимации старайтесь анимировать только те свойства, которые могут обрабатываться потоком компоновщика браузера, такие как `transform` и `opacity`. Анимация таких свойств, как `width` или `height` может привести к проблемам с производительностью.
- Используйте правильный порядок объявлений: объявляйте `animation-timeline` и `animation-range` после стандартного сокращения `animation` во избежание сброса временной шкалы сокращением.

При использовании функции `scroll()` для создания анимации, управляемой прокруткой:

- Опционально: явно указывайте прокручиваемого предка для отслеживания с помощью `scroll(root)` или `scroll(self)`.
  - Если `root`, `nearest` или `self` не подходят, используйте именованное `scroll-timeline`.
- Опционально: явно определяйте ось для отслеживания с помощью `scroll(block)` или `scroll(inline)`.

Конкретно для этого случая:

- Анимируемый элемент не должен быть `position: static` или `position: relative` при использовании процентов в `animation-range`.
  - Такие элементы считаются находящимися "в потоке" (in-flow). Сжатие таких элементов при прокрутке сжимает общее расстояние прокрутки, что влияет на вычисляемое значение, например, `10%` в прокрутке.

## Поддержка и страховка

Анимация, управляемая прокруткой, имеет ограниченную поддержку браузеров. Поддерживается в Chrome 115+, Edge 115+ (с июля 2023 года) и Safari 26+ (с сентября 2025 года). Не поддерживается в Firefox.

Для браузеров без поддержки можно использовать резерв для воссоздания визуальных эффектов. Резерв, как правило, создается с помощью обработчика прокрутки (для эффектов ScrollTimeline) или IntersectionObserver API (для эффектов ViewTimeline).

В браузерах с поддержкой всегда используйте нативную реализацию CSS, поскольку она гораздо более производительна.

Обратите внимание, что не все эффекты можно воссоздать с помощью резерва.

Следующий скрипт применяет резерв для браузеров, которые не поддерживают анимацию, управляемую прокруткой. В нем используется обработчик прокрутки, который отслеживает положение прокрутки документа на расстоянии `150px` и соответствующим образом обновляет высоту шапки:

```js
if (!CSS.supports('(animation-timeline: scroll()) and (animation-range: 0% 100%)')) {
  const header = document.querySelector('header');

  const initialHeight = 200;
  const finalHeight = 50;
  const scrollDistance = 150;

  window.addEventListener('scroll', () => {
    const scrollY = window.scrollY;
    const scrollPercent = Math.min(1, scrollY / scrollDistance);
    const newHeight = initialHeight - (initialHeight - finalHeight) * scrollPercent;

    header.style.height = `${newHeight}px`;
  });
}
```
