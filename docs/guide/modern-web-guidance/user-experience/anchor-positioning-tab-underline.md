# Подчеркивание активной вкладки с помощью якорного позиционирования

В меню вкладок вы должны предоставлять пользователям визуальные подсказки о том, на какой странице они находятся. Один из вариантов - подчеркнуть (underline) активную вкладку. Позиционирование якоря (anchor positioning) позволяет создать плавную анимацию между положениями подчеркивания. Это не работает, когда изменение активной вкладки загружает новую страницу.

Этот эффект можно также использовать для анимирования точки для индикации активной вкладки в вертикальном меню.

Создаем подчеркивание с помощью псевдоэлемента `::before` элемента `<ul>`, содержащего элементы `<li>`. Использование псевдоэлемента является предпочтительным подходом - в DOM не добавляются лишние чисто декоративные элементы.

```css
ul::before {
  /* Добавляем псевдоэлемент в контейнер для представления анимированного индикатора */
  content: '';
}
```

Делаем активный элемент списка якорем путем добавления свойства `anchor-name` со значением, начинающимся с `--`:

```css
li.active {
  /* Присваиваем уникальное anchor-name активному элементу */
  anchor-name: --active;
}
```

Привязываем подчеркивание к активному элементу с помощью свойства `position-anchor`, значение которого совпадает со значением свойства `anchor-name` якоря, и позиционируем его абсолютно:

```css
ul::before {
  /* Привязываем подчеркивание к активному элементу */
  position: absolute;
  position-anchor: --active;
}
```

Позиционируем подчеркивание относительно якоря с помощью свойств `inset` и функций `anchor()`:

```css
ul::before {
  /* Не используйте свойство "position-area", оно не анимируется. */
  /* Используем calc() для небольшого отступа сверху */
  inset-block-start: calc(anchor(bottom) + .1lh);
  inset-inline-start: anchor(left);
  inset-inline-end: anchor(right);
}
```

Добавляем высоту и другие визуальные стили:

```css
ul::before {
  block-size: .25lh;
  background: red;
}
```

Наконец, добавляем переход (transition) свойств `inset`:

```css
ul::before {
  @media (prefers-reduced-motion: no-preference) {
    /* Переход должен быть обернут в медиа-запрос prefers-reduced-motion для учета предпочтений пользователя */
    transition: inset .2s;
  }
}
```

Это только визуальный индикатор, который не заменяет соответствующие значения ARIA (такие как `aria-current="page"` или `aria-selected`).

```html
<!-- Предоставляем явное состояние для вспомогательных технологий -->
<nav aria-label="Primary">
  <ul>
    <li class="active">
      <a href="/home" aria-current="page">Домой</a>
    </li>
    <li>
      <a href="/about">Контакты</a>
    </li>
  </ul>
</nav>
```

## Поддержка и страховка

Anchor Positioning API поддерживается всеми основными браузерами (в Safari пока только в качестве экспериментальной возможности).

При отсутствии поддержки, используйте `border-bottom` для добавления подчеркивания. Оно не будет анимироваться.

```css
ul li.active {
  @supports not (position-anchor: auto) {
    border-bottom: .25lh var(--primary) solid;
  }
}
```
