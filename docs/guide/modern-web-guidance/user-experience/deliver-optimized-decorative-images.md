# Доставка оптимизированных декоративных изображений

Доставка оптимизированных декоративных изображений с помощью CSS улучшает воспринимаемую производительность без ухудшения визуального качества. Функция CSS `image-set()` позволяет предоставить браузеру несколько вариантов для одного фонового изображения или изображения маски (mask image). Можно определить современные форматы (такие как AVIF или WebP) вместе с разными разрешениями (такими как `1x` и `2x`). Браузер будет динамически выбирать наименьшее совместимое изображение, предоставляющее подходящую плотность пикселей для устройства пользователя.

Предупреждение: если изображение является элементом отрисовки наибольшего контента (LCP) (например, большим основным баннером), учитывайте, что изображение, указанное в CSS с помощью `image-set()`, не обнаруживается сканером предварительной загрузки браузера. Это может существенно увеличить время загрузки изображения и ухудшить LCP. Для кандидатов LCP старайтесь использовать стандартные теги HTML `<img>` или `<picture>`, также предварительно загружая изображение с помощью `<link rel=preload>` с атрибутом `media`.

## Реализация

Функция `image-set()` используется везде, где ожидается значение `<image>`, чаще всего, в `background-image`, `content` или `mask-image`. Обратите внимание, что хотя предоставление формата изображения с помощью `type()` и разрешения (такого как `1x` или `2x`) дает лучшие результаты, оба аргумента являются опциональными:

```css
.gallery-item {
  /* Предоставляем несколько разрешений и форматов с помощью image-set(). */
  /* Форматы всегда следует упорядочивать от наиболее оптимизированного (AVIF) к наименее (JPEG/PNG).
     Браузер остановится на первом поддерживаемом формате. */
  background-image: image-set(
    url("gallery.avif") type("image/avif") 1x,
    url("gallery-2x.avif") type("image/avif") 2x,
    url("gallery.webp") type("image/webp") 1x,
    url("gallery-2x.webp") type("image/webp") 2x,
    url("gallery.jpg") type("image/jpeg") 1x,
    url("gallery-2x.jpg") type("image/jpeg") 2x
  );

  /* Стандартные декоративные свойства */
  background-size: cover;
  background-position: center;
}
```

## Поддержка и страховка

Функция `image-set()` поддерживается всеми основными браузерами с 18.09.2023.

Для браузеров без поддержки следует предоставить стандартное объявление изображения перед `image-set()`. Эта стратегия прогрессивного улучшения основана на каскадной природе CSS: не поддерживаемые правила игнорируются.

```css
.gallery-item {
  /* Резерв для браузеров без поддержки */
  background-image: url("gallery.jpg");

  /* Современные браузеры применят это и перезапишут резерв */
  background-image: image-set(
    url("gallery.avif") type("image/avif") 1x,
    url("gallery-2x.avif") type("image/avif") 2x,
    url("gallery.jpg") type("image/jpeg") 1x,
    url("gallery-2x.jpg") type("image/jpeg") 2x
  );
}
```
