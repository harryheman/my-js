# Запрет переноса текста

Современный CSS предоставляет свойство `text-wrap` для управления тем, как текст переносится в пределах его контейнера. Для обеспечения того, чтобы текст оставался в одной строке и игнорировал границы контейнера, используйте `text-wrap: nowrap`. Это современная более семантическая замена `white-space: nowrap`.

Предотвращение переноса текста полезно для элементов UI, таких как вкладки навигации, горизонтально прокручиваемые чипсы (chips) или в любом сценарии, когда перенос строки ломает макет или визуальный дизайн.

## Реализация

### Базовое использование

Для предотвращения любых автоматических переносов текста примените `text-wrap: nowrap` к элементу, содержащему текст.

1. Примените `text-wrap: nowrap` к целевому элементу.
2. Опционально: используйте свойство `overflow` (со значением `hidden`, `scroll` или `auto`) для управления переполнением.
3. Опционально: используйте `text-overflow: ellipsis` для предоставления визуальной подсказки при обрезке текста. Обратите внимание: это требует установки `overflow: hidden`.

### Пример кода

```css
.no-wrap-text {
  /* Предотвращаем автоматический перенос текста */
  text-wrap: nowrap;

  /* Обрабатываем визуальное переполнение */
  overflow: hidden;
  text-overflow: ellipsis;

  /* Ограничиваем ширину для включения и обработки переполнения этого элемента */
  max-width: 200px;
}
```

### Дополнительный контроль с помощью длинных свойств

Свойство `text-wrap` - это сокращение для `text-wrap-mode` (переносится ли текст) и `text-wrap-style` (как он переносится). Поскольку `text-wrap-style` игнорируется при отключенном переносе, обычно, следует использовать сокращение `text-wrap`.

Хотя длинное `text-wrap-mode: nowrap` существует, название этого свойства рабочей группой CSS рассматривается как временное и может измениться в будущем:

```css
.granular-control {
  /* Современный длинный эквивалент `white-space: nowrap`. */
  /* Лучше использовать `text-wrap: nowrap;` */
  text-wrap-mode: nowrap;
}
```

### Поддержка и страховка

Свойство `text-wrap` поддерживается всеми основными браузерами с 17.10.2024.

В браузерах без поддержки можно использовать старое свойство `white-space`. Современные браузеры расценивают `white-space` как сокращение свойств `text-wrap-mode` и `white-space-collapse`:

```css
.no-wrap-with-fallback {
  /* Резерв для старых браузеров */
  white-space: nowrap;

  /* Современный стандарт */
  text-wrap: nowrap;
}
```
